# MBG Toys Web Scraper

## Overview
This project is a Python web scraper developed to extract product information from the MBG Toys e-commerce website using Selenium and BeautifulSoup.

## Features
- Extracts product names
- Extracts product prices
- Handles multiple pages
- Saves data into a CSV file

## Technologies
- Python
- Selenium
- BeautifulSoup
- Pandas

## Output
The scraper successfully extracted over 370 product records and exported them into a structured CSV file.

## Project Structure
```
mbg_scraper.py
output.csv
```
